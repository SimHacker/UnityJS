UnityJS/Assets/Libraries/UnityJS/link.xml
  Linker xml configuration file, which declares which assemblies
  should have their symbols preserved in order for reflection to work.

  If you get runtime errors about undefined names on a Unity or third
  party class that should be defined, add a declaration of its
  assembly with preserve="all" to this file.
